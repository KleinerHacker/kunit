KSpeedUnit

enum KSpeedUnit : KUnit, Enum<KSpeedUnit>

Enumerates concrete units of speed (a constructed quantity: length · time⁻¹). baseValue is the factor to convert into the group's base unit (BASE, meter per second): 1 unit = baseValue * m/s.

KSpeedUnitInstance
Link copied to clipboard
class KSpeedUnitInstance : KUnitInstance<KSpeedUnitInstance> , KUnitMeasurable

Wraps a KMixedUnitInstance representing a speed, i.e. exactly two terms - one KDistanceUnit.BASE (meter) at exponent +1 and one KTimeUnit.BASE (second) at exponent -1 (m·s⁻¹). The value is always normalized internally to meters per second (KSpeedUnit.BASE), regardless of which KSpeedUnit, SI org.pcsoft.framework.kunit.KUnitPrefix, or length/time combination it was constructed from.

toSpeed
Link copied to clipboard
fun KMixedUnitInstance.toSpeed(): KSpeedUnitInstance

Converts this mixed unit to a "pure" speed value, as long as it consists of exactly two terms: one KDistanceUnit term at exponent +1 and one KTimeUnit term at exponent -1 (order independent). The terms are normalized to KDistanceUnit.BASE/KTimeUnit.BASE, so the resulting speed is expressed in meters per second regardless of which length/time units the terms were tagged with (e.g. a [MILE^1, HOUR^-1] instance is converted to the equivalent m/s).
