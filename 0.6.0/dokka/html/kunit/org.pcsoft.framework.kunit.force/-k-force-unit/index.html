kunit/org.pcsoft.framework.kunit.force/KForceUnit

KForceUnit

enum KForceUnit : KUnit, Enum<KForceUnit>

Enumerates concrete units of force (a constructed quantity: mass · length · time⁻²). baseValue is the factor to convert into the group's base unit (BASE, newton): 1 unit = baseValue * N.

Force is a constructed, three-term unit (kg·m·s⁻²). Because the mass component of this library is normalized to grams (not kilograms), the newton is 1000× the raw component base (g·m·s⁻²); that fixed factor is org.pcsoft.framework.kunit.force.N_IN_BASE and is applied by the KForceUnitInstance factory, not by these baseValues (which stay relative to the newton).

The kilopond (kp) / kilogram-force (kgf) is intentionally not a dedicated entry: it is reached generically through the SI prefix table applied to POND (kilo.ponds), just like the kilonewton is kilo.newtons.

Example:

KForceUnit.POND.baseValue // 0.009 806 65 (1 p = 9.806 65 mN)
