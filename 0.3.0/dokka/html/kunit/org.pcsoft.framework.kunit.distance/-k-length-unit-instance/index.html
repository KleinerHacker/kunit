KLengthUnitInstance

class KLengthUnitInstance : KDistanceUnitInstance, KUnitInstance<KLengthUnitInstance>

A length: the exponent-1 leaf of the distance group, a single KDistanceUnit.BASE term at exponent 1. Being its own type (distinct from the general KDistanceUnitInstance) is what gives the compile-time guarantees:

  • length + length, length - length, and the comparison operators are typed to KLengthUnitInstance and therefore only accept another length - length + area does not compile.

  • length * length = area, length * area = volume, area / length = length, etc. are strongly typed (see the times/div overloads), while a length combined with the general KDistanceUnitInstance or a raw KMixedUnitInstance falls back to a KMixedUnitInstance.

Instances are created via the creator extension properties in KLengthUnitExtensions.kt (e.g. 5.meters, 5.miles) or the SI-prefix infix constructors in KDistanceUnitPrefix.kt (e.g. 3 kilo meters).

Example:

val d = 5.miles
d.value                       // 8046.72 (normalized to meters)
d.valueAs(KDistanceUnit.MILE) // 5.0
val area = 200.meters * 50.meters // KAreaUnitInstance, 10000 m²
