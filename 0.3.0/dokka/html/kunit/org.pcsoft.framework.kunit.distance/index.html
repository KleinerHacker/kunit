KAreaUnitInstance

class KAreaUnitInstance : KDistanceUnitInstance, KUnitInstance<KAreaUnitInstance>

An area: the exponent-2 leaf of the distance group, a single KDistanceUnit.BASE term at exponent 2 (square meters). Like KLengthUnitInstance it is its own type, so area + area / comparisons only accept another area (area + length does not compile), while *// are strongly typed (e.g. area / length = length, area * length = volume).

KDistanceDerivedUnit
Link copied to clipboard
object KDistanceDerivedUnit

Named units for length-derived physical quantities (area, volume), each bound to a fixed exponent of KDistanceUnit.BASE. Usable as a KUnitTarget wherever a plain KDistanceUnit or KScaledUnit would be accepted, e.g. with KMixedUnitInstance.valueAs/toString.

KDistanceUnit
Link copied to clipboard
enum KDistanceUnit : KUnit, Enum<KDistanceUnit>

Enumerates concrete units of length. baseValue is the factor to convert into the group's base unit (BASE, meter): 1 unit = baseValue * meter.

class KVolumeUnitInstance : KDistanceUnitInstance, KUnitInstance<KVolumeUnitInstance>

A volume: the exponent-3 leaf of the distance group, a single KDistanceUnit.BASE term at exponent 3 (cubic meters). Like the other leaves it is its own type, so volume + volume / comparisons only accept another volume, while *// are strongly typed (e.g. volume / area = length, volume / length = area).

infix fun KDistanceUnitInstance.pow(n: Int): KDistanceUnitInstance

Raises this distance value to the integer power n, staying inside the distance hierarchy: the normalized KDistanceUnitInstance.value is raised to n and the KDistanceUnitInstance.exponent is multiplied by n, so the concrete result type follows the resulting exponent (KLengthUnitInstance/KAreaUnitInstance/KVolumeUnitInstance for 1/2/3, otherwise the general KDistanceUnitInstance).

toDistance
Link copied to clipboard
fun KMixedUnitInstance.toDistance(): KDistanceUnitInstance

Converts this mixed unit to a "pure" distance value (length, area, volume, or any other exponent), as long as it consists of exactly one term of any KDistanceUnit, at any exponent - normalizing it to KDistanceUnit.BASE if it isn't already. The concrete runtime type is the matching leaf (KLengthUnitInstance/KAreaUnitInstance/KVolumeUnitInstance) for exponent 1/2/3, or the general KDistanceUnitInstance otherwise.
