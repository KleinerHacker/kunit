KScaledUnit

data class KScaledUnit(val prefix: KUnitPrefix, val unit: KUnit) : KUnitTarget

A KUnit combined with a KUnitPrefix, e.g. "km" = KILO with METER.

Used as a KUnitTarget wherever a plain KUnit would otherwise be accepted (e.g. KLengthUnitInstance.valueIn, KUnitInstance.valueAs/toString), so callers can pass either a bare unit or a prefixed one at the same call site.

Example:

val km = KUnitPrefix.KILO with KLengthUnit.METER
km.baseValue // 1000.0 (1 km = 1000 m)
km.symbol    // "km"
