KLengthDerivedUnit

object KLengthDerivedUnit

Named units for length-derived physical quantities (area, volume), each bound to a fixed exponent of KLengthUnit.BASE. Usable as a KUnitTarget wherever a plain KLengthUnit or KScaledUnit would be accepted, e.g. with KUnitInstance.valueAs/toString.

KLengthUnit
Link copied to clipboard
enum KLengthUnit : KUnit, Enum<KLengthUnit>

Enumerates concrete units of length. baseValue is the factor to convert into the group's base unit (BASE, meter): 1 unit = baseValue * meter.

KLengthUnitInstance
Link copied to clipboard
class KLengthUnitInstance

Wraps a KUnitInstance with exactly one term of KLengthUnit.BASE (meter), at any exponent - exponent 1 for a "pure" length, 2 for an area, 3 for a volume, etc. The value is always normalized internally to KLengthUnit.BASE, regardless of which unit or org.pcsoft.framework.kunit.KUnitPrefix it was constructed with.

toKLengthUnit
Link copied to clipboard
fun KUnitInstance.toKLengthUnit(): KLengthUnitInstance

Converts this mixed unit to a "pure" length-based value (length, area, volume, ...), as long as it consists of exactly one term of any KLengthUnit, at any exponent - normalizing it to KLengthUnit.BASE if it isn't already (e.g. a term tagged with KLengthUnit.MILE is converted to the equivalent value in meters). This is what lets org.pcsoft.framework.kunit.KPrefixBuilder (which, being generic, tags terms with whichever org.pcsoft.framework.kunit.KUnit it was given, not necessarily the group's base unit) be converted into a KLengthUnitInstance.
