KAreaUnitInstance

class KAreaUnitInstance : KDistanceUnitInstance, KUnitInstance<KAreaUnitInstance>

An area: the exponent-2 leaf of the distance group, a single KDistanceUnit.BASE term at exponent 2 (square meters). Like KLengthUnitInstance it is its own type, so area + area / comparisons only accept another area (area + length does not compile), while *// are strongly typed (e.g. area / length = length, area * length = volume).

Instances are created via the power operation on a length (e.g. 200.meters pow 2, 5 kilo meters pow 2), the derived-unit creator extension properties in KAreaUnitExtensions.kt (e.g. 5.hectares, 5.ares, 1.acres), or as the result of length * length.

Example:

val a = 200.meters * 50.meters // 10000 m²
a.valueAs(KDistanceDerivedUnit.HECTARE) // 1.0
(a / 100.meters).valueAs(KDistanceUnit.METER) // 100.0 (an area divided by a length is a length)
