KLengthDerivedUnit

object KLengthDerivedUnit

Named units for length-derived physical quantities (area, volume), each bound to a fixed exponent of KLengthUnit.BASE. Usable as a KUnitTarget wherever a plain KLengthUnit or KScaledUnit would be accepted, e.g. with KMixedUnitInstance.valueAs/toString.

KLengthUnit
Link copied to clipboard
enum KLengthUnit : KUnit, Enum<KLengthUnit>

Enumerates concrete units of length. baseValue is the factor to convert into the group's base unit (BASE, meter): 1 unit = baseValue * meter.

KLengthUnitInstance
Link copied to clipboard
class KLengthUnitInstance : KUnitInstance<KLengthUnitInstance> , KUnitMeasurable

Wraps a KMixedUnitInstance with exactly one term of KLengthUnit.BASE (meter), at any exponent - exponent 1 for a "pure" length, 2 for an area, 3 for a volume, etc. The value is always normalized internally to KLengthUnit.BASE, regardless of which unit or org.pcsoft.framework.kunit.KUnitPrefix it was constructed with.

toKLengthUnit
Link copied to clipboard
fun KMixedUnitInstance.toKLengthUnit(): KLengthUnitInstance

Converts this mixed unit to a "pure" length-based value (length, area, volume, ...), as long as it consists of exactly one term of any KLengthUnit, at any exponent - normalizing it to KLengthUnit.BASE if it isn't already (e.g. a term tagged with KLengthUnit.MILE is converted to the equivalent value in meters). This is what lets an arbitrary single-KLengthUnit mixed instance (whose term may be tagged with any KLengthUnit, not necessarily the group's base unit) be converted into a KLengthUnitInstance.
