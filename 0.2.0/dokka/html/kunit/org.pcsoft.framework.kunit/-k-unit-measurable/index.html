KUnitMeasurable

interface KUnitMeasurable

The common abstraction shared by every unit value in this library: the generic mixed unit (KMixedUnitInstance) as well as every "pure" unit wrapper (e.g. KLengthUnitInstance, KTimeUnitInstance).

It captures exactly the surface that is identical across all of them:

  • a normalized value (always a Double),

  • a conversion into the generic KMixedUnitInstance engine (toKMixedUnitInstance),

  • multiplication and division against an arbitrary KMixedUnitInstance (times/div).

The "pure" wrappers do not implement these members by hand - they delegate this interface to their underlying KMixedUnitInstance via Kotlin's by delegation (e.g. class KLengthUnitInstance(...) : KUnitInstance<KLengthUnitInstance>, KUnitMeasurable by instance). Only KMixedUnitInstance implements it directly.

The richer, self-typed surface of the "pure" wrappers (same-type +/-/comparison, unit-target conversion) lives in the sub-interface KUnitInstance.

Example:

val values: List<KUnitMeasurable> = listOf(5.meters, 2.hours, 10.meters / 2.seconds)
values.map { it.value }                 // normalized base values
values.map { it.toKMixedUnitInstance() } // uniform generic representation
