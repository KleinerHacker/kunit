# درجة الحرارة — نظرة عامة

الحزمة: `org.pcsoft.framework.kunit.temperature`

تُنمذَج درجة الحرارة بـ**مجموعتين مترابطتين**، لأنّ قراءة درجة الحرارة و*تغيّرها* كمّيتان مختلفتان
فيزيائيًا. وضبط هذا التمييز هو ما يجعل الحساب صحيحًا.

| المفهوم | النوع | الطبيعة | الوحدة الأساسية |
|---|---|---|---|
| [درجة الحرارة المطلقة](temperature.md) | `KTemperatureUnitInstance` | **نقطة** أفينية | كلفن (`K`) |
| [فرق درجة الحرارة](temperature-difference.md) | `KTemperatureDifferenceUnitInstance` | **فترة** خطّية | كلفن (`ΔK`) |

## نقطة مقابل فترة

**درجة الحرارة المطلقة** (`25 °C`، `300 K`) هي *نقطة* أفينية على مقياس الحرارة: تُقاس من صفر ثابت،
واختيار الصفر (0 K مقابل 0 °C) يُزيح الرقم. أمّا **فرق درجة الحرارة** (`20 ΔK`) فهو *متّجه* — الفجوة بين
نقطتين. لا يحمل عبء نقطة الصفر: فرق قدره `20 ΔK` يساوي فرقًا قدره `20 °C`، بصرف النظر عن موضعه على المقياس.

هذا هو تمييز النقطة/المتّجه في فضاء أفيني بالضبط، وهو يحكم الحساب:

```kotlin
import org.pcsoft.framework.kunit.of
import org.pcsoft.framework.kunit.temperature.*

val a = 30 of celsius
val b = 10 of celsius

a - b                                    // KTemperatureDifferenceUnitInstance: 20 ΔK  (فترة)
a + KTemperatureDifference.ofKelvin(5)   // KTemperatureUnitInstance: 308.15 K         (ما زالت نقطة)
// a + b                                 // لا يُصرَّف — جمع نقطتين بلا معنى
```

| العملية | النتيجة |
|---|---|
| `درجة مطلقة − درجة مطلقة` | **فرق درجة الحرارة** (فترة كلفن) |
| `درجة مطلقة + فرق` | درجة الحرارة المطلقة |
| `درجة مطلقة − فرق` | درجة الحرارة المطلقة |
| `فرق ± فرق` | فرق درجة الحرارة |
| `درجة مطلقة + درجة مطلقة` | **خطأ تصريف** (بلا معنى فيزيائي) |

## لماذا يهمّ فيزيائيًا

يعتمد ما إذا كانت صيغة تريد درجة حرارة مطلقة أم فرقًا على **كيف** تظهر درجة الحرارة:

* **كفرق / تغيّر** ← استخدم **فرق درجة الحرارة**. تُلغى نقطة الصفر، فتصبح `°C` و`K` قابلتين للتبادل هنا.
  الحالة الكلاسيكية — الطاقة الحرارية:

  $$Q = m \cdot c \cdot \Delta T$$

  `30 °C − 10 °C = 20 ΔK`، و`Q = m·c·20 ΔK`. كذلك التمدّد الحراري (`ΔL = α·L·ΔT`)، والتوصيل، ووحدات
  الحرارة النوعية / التوصيل الحراري (`J/(kg·ΔK)`، `W/(m·ΔK)`).

* **ضربيًا** (T وحدها، أو كقوّة `T⁴`، أو كنسبة `T₁/T₂`) ← استخدم **درجة حرارة مطلقة بالكلفن**، لأنّ الصفر
  المطلق جزء من الفيزياء: قانون الغاز المثالي `pV = nRT`، إشعاع ستيفان–بولتزمان `P = εσA·T⁴`، كفاءة
  كارنو `η = 1 − T_c/T_h`.

!!! warning "البُعد نفسه، كمّية مختلفة"
    كلتا المجموعتين لهما البُعد *كلفن*، فعلى مستوى الوحدة النقيّة يبدو `m·K` و`m·ΔK` متشابهين. لكنّهما
    **ليسا** الوحدة نفسها هنا: تستخدمان مجموعتَي وحدات متمايزتين، فوحدة مختلطة تحوي كلفن مطلقًا ليست مساويةً
    ولا قابلةً للجمع مع أخرى تحوي كلفن فرق. ويُطبَع الرمز المميّز **`ΔK`** (مقابل `K`) تحديدًا لإظهار ذلك
    في `toString` وفي مخرجات الوحدات المختلطة — فضَعه في حسبانك عند تأليف وحداتك المختلطة.
